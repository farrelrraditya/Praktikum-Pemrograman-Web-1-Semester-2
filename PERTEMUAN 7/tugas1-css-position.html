Tentang CSS Position

CSS Position digunakan untuk mengatur tata letak elemen di halaman web. Ada beberapa nilai: static, relative, absolute, fixed, dan sticky. Masing-masing punya perilaku yang berbeda tergantung konteks penggunaannya.

Position Relative

Elemen dengan position: relative tetap berada di alur normal dokumen, tapi bisa digeser menggunakan top, right, bottom, left. Elemen ini juga jadi parent/acuan bagi elemen absolute di dalamnya.

Position Absolute

Elemen absolute dikeluarkan dari alur normal dokumen dan diposisikan relatif terhadap ancestor terdekat yang punya position selain static. Cocok untuk badge, tooltip, dropdown menu.

Position Fixed

Fixed membuat elemen menempel pada viewport (layar browser), tidak bergerak saat halaman di-scroll. Biasa dipakai untuk navbar, tombol back-to-top, atau chat widget.

Position Sticky

Sticky adalah kombinasi relative dan fixed. Elemen berperilaku seperti relative sampai mencapai titik tertentu saat scroll, lalu "menempel" seperti fixed. Cocok untuk header tabel atau navbar yang harus tetap terlihat.

Z-Index

Z-index mengontrol urutan tumpukan (stacking order) elemen yang saling overlap. Nilai lebih tinggi berarti elemen tampil di depan. Z-index hanya berlaku pada elemen yang punya position selain static.
